# SPHAR Dataset

In this research, we use SPHAR dateset. The SPHAR (Surveillance Perspective Human Action Recognition) dataset is designed for human action recognition tasks. It contains 14 action classes such as hitting, kicking, falling, vandalizing, panicking, sitting, walking, running, neutral, luggage, stealing, murdering, carcrash, and igniting. The dataset includes diverse scenarios, varying poses, and action types, making it suitable for training and evaluating machine learning models in human action recognition and anomaly detection.

- **Format**: Video sequences.
- **Annotations**: 14 classes.
- **Applications**: Human action recognition, anomaly detection, and behavior analysis.

For more details, refer to the dataset documentation.

### Classification of Events

The 14 action classes in the SPHAR dataset can be categorized into normal and abnormal events as follows:

#### Normal Events
- Sitting
- Walking
- Running
- Neutral
- Luggage

#### Abnormal Events
- Hitting
- Kicking
- Falling
- Vandalizing
- Panicking
- Stealing
- Murdering
- Carcrash
- Igniting
